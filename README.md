# ML4space23

Repository for [Summer School ML4Space](https://sites.google.com/view/ml4space/home).

# Installation

### With Anaconda

We recommend the use of Anaconda during the Summer School. 

**Step 1** Download Anaconda [here](https://www.anaconda.com/download/) and follow the installation instructions on the website. 

**Step 2** Create a dedicated environment for the Summer School with Python 3. Instructions on how to create an environment can be found [here](https://docs.anaconda.com/free/navigator/tutorials/manage-environments/) 

**Step 3** In your environment, please install the following
- Pytorch with CUDA ([instructions](https://pytorch.org/get-started/locally/))

### With Colab

Alternatively, you can use [Colab](https://colab.research.google.com/) through your Google account
